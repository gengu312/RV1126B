## 操作前检查

- [ ] 开发板型号和原理图版本一致。
- [ ] 物理针脚和网络名已由排针图、原理图两处交叉核对。
- [ ] GPIO 没有被屏幕、摄像头、音频、电源、复位或其他驱动占用。
- [ ] GPIO 电平已确认，外部 LED 串有限流电阻。
- [ ] ADC 最大输入电压已确认，输入信号与开发板共地。
- [ ] 未把 5V、负电压或未知信号直接接入 GPIO/ADC。
